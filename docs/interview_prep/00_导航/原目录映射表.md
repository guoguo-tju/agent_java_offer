# 资料重组映射说明

本公开版资料库采用“方向 -> 主题 -> 单文件问答”的结构。

## 重组原则

1. 原始资料按主题重新归并，而不是按原文件名保留。
2. 同一主题下尽量把主问题、追问、口述版和补充材料收敛到一个文件中。
3. 公开版不再保留私人工作区路径和本地文件定位信息。

## 映射关系

- AI 相关原始资料 -> `01_AI/`
- 后端题库与补充资料 -> `02_后端/`
- 系统设计专题与补充讲稿 -> `03_系统设计/`
- 算法题型与题解思路 -> `04_算法/`
- 项目讲稿、追问与项目表达资料 -> `05_项目表达/`
- 公司/场景化面试题整理 -> `06_公司实战/`

## 说明

如果后续继续开源清洗，这份文件只保留重组逻辑说明，不再回指私人原始目录。
