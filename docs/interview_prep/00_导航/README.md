# interview_prep_v2 导航

这套目录是面向复习与面试表达的新版本资料层，目标是把原来按来源、公司、主题混排的内容，整理成：

**方向 -> 主题 -> 单文件问答**

## 设计原则

1. 原目录不动，`interview_prep_v2/` 只做副本整理。
2. 一个主题默认只保留一个主文件：`01_核心问答.md`。
3. 文件内部采用“一问一答”结构，并保留来源。
4. 允许把原来一个大文件中的不同片段拆到多个主题中。
5. 新目录以“复习可读性”优先，不强求和原目录一一同构。

## 当前进度

- 已完成：AI、后端、系统设计、算法、项目表达、公司实战六大方向的第一版整理
- 后续工作：持续补题、补口述版、补来源引用，而不是再调整目录结构
