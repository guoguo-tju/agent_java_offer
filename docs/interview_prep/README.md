# interview_prep

这是这套面试资料库的主入口。
目标不是堆零散题目，而是把 Java 后端、AI Agent、系统设计、算法和项目表达资料重组为**适合复习、口述和追问展开**的结构。

## 先看什么

如果你第一次进入这套资料，建议按这个顺序：

1. 先看 [00_导航/README.md](00_导航/README.md)
2. 再看 [总索引](00_导航/总索引.md)
3. 最后进入你当前最需要的方向目录

如果你时间有限，最短路径是：

- Java 后端转 Agent：先看 `01_AI`，再补 `02_后端`
- 后端面试冲刺：先看 `02_后端`，再刷 `03_系统设计`
- 项目口述强化：直接进 `05_项目表达`

## 这套资料怎么组织

统一按下面的结构组织：

**方向 -> 主题 -> 单文件问答**

也就是说：

- 先按大方向找资料
- 再按主题进入子目录
- 每个主题主要看 `01_核心问答.md`

这样做的目的，是把原来“按来源、按公司、按零散问题混排”的内容，整理成更适合复习和面试表达的结构。
